# Лабораторная работа: Универсальный калькулятор

## Основная часть

### Общая идея проекта

В ходе работы было разработано приложение «Универсальный калькулятор» для платформы Windows. Архитектура приложения построена на принципах объектно-ориентированного программирования и разделяет ответственность между несколькими классами. Ключевая особенность проекта — поддержка трёх режимов работы с числами:

- **Действительные числа в системе счисления с основанием p (2–16)**
- **Рациональные дроби**
- **Комплексные числа в системе счисления с основанием p (2–16)**

Разработка велась на языке C# с использованием .NET 8.0 и Windows Forms для интерфейса.

### Иерархия чисел

#### Абстрактный класс TANumber

Базовый абстрактный класс для всех числовых типов. Определяет интерфейс, который должны реализовать все производные классы: арифметические операции (`Add`, `Subtract`, `Multiply`, `Divide`), функции (`Sqr`, `Reverse`), сравнение (`EqZero`, `EqualsNumber`), преобразование в строку (`ReadNumberAsString`, `WriteNumberFromString`) и определение типа (`GetType`).

```csharp
public abstract class TANumber
{
    protected string _stringValue;

    public abstract double ToDouble();
    public abstract bool EqZero();
    public abstract TANumber Copy();
    public abstract TANumber Add(TANumber other);
    public abstract TANumber Subtract(TANumber other);
    public abstract TANumber Multiply(TANumber other);
    public abstract TANumber Divide(TANumber other);
    public abstract bool EqualsNumber(TANumber other);
    public abstract TANumber Sqr();
    public abstract TANumber Reverse();
    public abstract string ReadNumberAsString();
    public abstract void WriteNumberFromString(string value);
    public abstract NumberType GetType();
}
```

#### Класс TPNumber (P-ичные числа)

Представляет действительные числа в системе счисления с основанием от 2 до 16. Хранит значение как `double` и текущее основание системы счисления. Поддерживает конвертацию из/в строку для любого допустимого основания, используя цифры `0–9` и `A–F`.

Основные возможности:
- хранение числового значения (`double`) и основания системы счисления (`int`)
- автоматическая конвертация при вводе/выводе для систем 2–16
- полный набор арифметических операций с числами других типов (Frac, TComplex)
- парсинг строки с учётом разделителя целой и дробной части

```csharp
public class TPNumber : TANumber
{
    private double _value;
    private int _baseSystem;

    public TPNumber(double value, int baseSystem)
    {
        _value = value;
        _baseSystem = Math.Max(2, Math.Min(16, baseSystem));
    }

    public override string ReadNumberAsString() =>
        ConvertToBaseStringInternal(_value, _baseSystem);
}
```

#### Класс Frac (Рациональные дроби)

Представляет простые дроби в виде пары целых чисел — числителя и знаменателя. Автоматически сокращает дробь до несократимого вида с помощью алгоритма Евклида.

Основные возможности:
- хранение числителя и знаменателя типа `long`
- автоматическое сокращение дробей
- арифметические операции с дробями и другими типами чисел
- разделитель — символ `/`

```csharp
public class Frac : TANumber
{
    private long _numerator;
    private long _denominator;

    public Frac(long numerator, long denominator)
    {
        _numerator = numerator;
        _denominator = denominator;
        Normalize(); // сокращение через НОД
    }
}
```

#### Класс TComplex (Комплексные числа)

Представляет комплексные числа в системе счисления с основанием p. Состоит из действительной и мнимой частей, каждая из которых является объектом `TPNumber`.

Основные возможности:
- хранение действительной и мнимой частей
- формат строкового представления: `a + bi*` или `a - bi*`
- арифметика комплексных чисел по правилам поля комплексных чисел
- вычисление модуля, сопряжённого числа, обратного числа

```csharp
public class TComplex : TANumber
{
    private TANumber _realPart;
    private TANumber _imaginaryPart;

    public TComplex(double real, double imag)
    {
        _realPart = new TPNumber(real);
        _imaginaryPart = new TPNumber(imag);
    }
}
```

### Иерархия редакторов

#### Абстрактный класс AEditor

Базовый класс для всех редакторов чисел. Определяет общий интерфейс: добавление цифры, разделителя, смена знака, backspace, очистка. Использует константы для команд редактирования.

```csharp
public abstract class AEditor
{
    public const uint cZero = 0;
    public const uint cSeparatorFR = 11;
    public const uint cSeparatorFrac = 13;
    public const uint cSeparatorComplex = 14;
    public const uint cBS = 15;

    public abstract string AddDigit(uint a);
    public abstract string AddSeparator(uint a = 0);
    public abstract string AddSigne(uint a = 0);
    public abstract string BackSpace();
    public abstract string Clear();
}
```

#### Класс REEditor (Редактор p-ичных чисел)

Редактор для ввода чисел в системах счисления 2–16. Отслеживает текущее основание, валидирует цифры (не допускает цифру `B` в двоичной системе), блокирует ведущие незначащие нули.

Особенности:
- поддержка цифр `0–9`, `A–F` в зависимости от основания
- блокировка ввода незначащих нулей (`0005` → `5`)
- автоматическая подстановка `0` перед разделителем при пустом поле

```csharp
internal class REEditor : AEditor
{
    private int _baseSystem;
    private bool _isNegative;

    public override string AddDigit(uint a)
    {
        if (a >= _baseSystem) return GetResultString();
        // блокировка ведущих нулей
        if (a == 0 && (string.IsNullOrEmpty(_stringValue) || _stringValue == "0"))
            return GetResultString();
        if (a != 0 && _stringValue == "0")
            _stringValue = GetDigitChar((int)a).ToString();
        else
            _stringValue += GetDigitChar((int)a);
        return GetResultString();
    }
}
```

#### Класс FEditor (Редактор дробей)

Редактор для ввода рациональных дробей в формате `числитель/знаменатель`. Отслеживает положение разделителя `/` и блокирует ведущие нули в обеих частях.

```csharp
internal class FEditor : AEditor
{
    private string _number;
    private bool _fractionSeparatorExists;
    private bool _inFractionPart;

    public override string AddSeparator(uint a = 0)
    {
        if (string.IsNullOrEmpty(_number)) _number = "0";
        _number += "/";
        _fractionSeparatorExists = true;
        _inFractionPart = true;
        return GetResultString();
    }
}
```

#### Класс CEditor (Редактор комплексных чисел)

Редактор для ввода комплексных чисел. Состоит из двух вложенных `REEditor` — для действительной и мнимой частей. Переключение между частями происходит по нажатию кнопки `i*`.

```csharp
internal class CEditor : AEditor
{
    private REEditor _realEditor;
    private REEditor _imagEditor;
    private bool _hasImaginaryPart;
}
```

#### Класс TEditor (Фасад редактора)

Обёртка, которая создаёт нужный редактор (`REEditor`, `FEditor` или `CEditor`) в зависимости от текущего типа числа и основания системы счисления. Предоставляет единый интерфейс для контроллера.

```csharp
public class TEditor
{
    private AEditor _editor;
    private NumberType _numberType;
    private int _baseSystem;

    public TEditor(NumberType numberType, int baseSystem)
    {
        _numberType = numberType;
        _baseSystem = baseSystem;
        CreateEditor();
    }

    private void CreateEditor()
    {
        _editor = _numberType switch
        {
            NumberType.Real => new REEditor(_baseSystem),
            NumberType.Fraction => new FEditor(),
            NumberType.Complex => new CEditor(_baseSystem),
            _ => new REEditor(_baseSystem)
        };
    }
}
```

### Класс TProc (Процессор)

Процессор выполняет арифметические операции и математические функции над объектами `TANumber`. Благодаря полиморфизму один и тот же процессор работает со всеми тремя типами чисел.

Обязанности:
- выполнение операций `+`, `−`, `×`, `÷`
- выполнение функций `sin`, `cos`, `tan`, `√x`, `log₁₀`, `ln`, `eˣ`, `|x|`, `x²`, `1/x`
- хранение левого и правого операндов
- обработка ошибок (деление на ноль, корень из отрицательного и т.д.)

```csharp
public class TProc
{
    private TANumber _lopd_Res;
    private TANumber _ropd;
    private TOperation _operation;
    private string _error;

    public void OprtnRun()
    {
        _lopd_Res = _operation switch
        {
            TOperation.Add => _lopd_Res.Add(_ropd),
            TOperation.Subtract => _lopd_Res.Subtract(_ropd),
            TOperation.Multiply => _lopd_Res.Multiply(_ropd),
            TOperation.Divide => _lopd_Res.Divide(_ropd),
            _ => _lopd_Res
        };
    }

    public void FuncRun(TFunction func)
    {
        _lopd_Res = func switch
        {
            TFunction.Sqr => _lopd_Res.Sqr(),
            TFunction.Reverse => _lopd_Res.Reverse(),
            TFunction.Sqrt => Sqrt(_lopd_Res),
            _ => _lopd_Res
        };
    }
}
```

### Класс TMemory (Память)

Хранит объект `TANumber` и предоставляет операции `MS`, `MR`, `MC`, `M+`, `M−`. Благодаря полиморфизму память работает с любым типом чисел.

```csharp
public class TMemory
{
    private TANumber _storedValue;
    private bool _hasValue;

    public void Store(TANumber number) => _storedValue = number.Copy();
    public TANumber Recall() => _storedValue?.Copy();
    public void Add(TANumber number) => _storedValue = _storedValue.Add(number);
    public void Clear() { _storedValue = null; _hasValue = false; }
}
```

### Класс TCtrl (Управление калькулятором)

Центральный контроллер, координирующий работу редактора, процессора, памяти и буфера обмена. Распределяет команды пользователя между компонентами и отслеживает состояние калькулятора.

Особенности реализации:
- поддержка команд для цифр `0–9` и `A–F`
- поддержка разделителей: точки (для p-ичных), `/` (для дробей), `i*` (для комплексных)
- корректное обновление дисплея после функций и операций через `_editor.SetNumberWithoutReset()`
- динамическое создание чисел нужного типа при смене режима

```csharp
public class TCtrl
{
    public const int CMD_DIGIT_0 = 0;
    public const int CMD_DIGIT_A = 35;
    public const int CMD_SEPARATOR_FRAC = 41;
    public const int CMD_SEPARATOR_COMPLEX = 42;
    public const int CMD_FUNC_SQR = 43;
    public const int CMD_FUNC_REV = 44;

    public string ExecuteFunction(int command)
    {
        _number = _editor.GetNumber();
        _processor.Lopd_Res = _number.Copy();
        _processor.FuncRun(func);
        _number = _processor.Lopd_Res.Copy();
        _editor.SetNumberWithoutReset(_number);
        return _editor.GetDisplayString();
    }
}
```

### Класс CalculatorForm (Интерфейс)

Графический интерфейс, реализованный на Windows Forms. Основные особенности:

- **Увеличенные размеры**: окно ~1400×1400 пикселей, кнопки 150×80 пикселей
- **Динамическая видимость кнопок**: при выборе основания `p` видны только цифры `0` до `p−1`
- **ComboBox для выбора основания**: значения от 2 до 16
- **Меню «Тип числа»**: переключение между p-ичными, дробями и комплексными
- **Индикатор режима**: отображает текущий тип числа и основание системы счисления
- **Тёмная тема**: цветовое кодирование кнопок (серые — цифры, оранжевые — операции, синие — функции, красные — память)

```csharp
public class CalculatorForm : Form
{
    private void UpdateButtonVisibility()
    {
        _btn0.Visible = _currentBaseSystem > 0;
        _btn1.Visible = _currentBaseSystem > 1;
        // ...
        _btnF.Visible = _currentBaseSystem > 15;
    }

    private void SetNumberType(NumberType type, int baseSystem)
    {
        _controller.Editor = new TEditor(type, baseSystem);
        _controller.Number = type switch
        {
            NumberType.Real => new TPNumber(0, baseSystem),
            NumberType.Fraction => new Frac(0, 1),
            NumberType.Complex => new TComplex(0, 0),
            _ => new TPNumber(0, baseSystem)
        };
    }
}
```

## Выводы

В ходе лабораторной работы был разработан универсальный калькулятор на языке C# с графическим интерфейсом Windows Forms. Приложение поддерживает три типа чисел — действительные числа в системах счисления 2–16, рациональные дроби и комплексные числа. Архитектура построена на полиморфизме: абстрактные классы `TANumber` и `AEditor` позволяют процессору, памяти и контроллеру работать с любым типом чисел единообразно. Реализована динамическая видимость кнопок цифр в зависимости от выбранного основания системы счисления, блокировка незначащих нулей и система контекстной справки.
