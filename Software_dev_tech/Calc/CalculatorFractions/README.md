# Калькулятор простых дробей

## Описание
Калькулятор для выполнения вычислений над простыми дробями с графическим интерфейсом.
Реализован в соответствии с заданием на объектно-ориентированное программирование C++.

## Архитектура проекта

### Модули

| Модуль | Класс | Описание |
|--------|-------|----------|
| UFrac | TFrac | Абстрактный тип данных "Простая дробь" |
| UEditor | TEditor | Ввод и редактирование простых дробей |
| UMemory | TMemory<T> | Параметризованная память для хранения значений |
| UProc | TProc<T> | Параметризованный процессор для выполнения операций |
| UControl | TCtrl | Управление калькулятором |
| UClcPnl | TClcPnl | Интерфейс (форма VCL для C++ Builder) |

## Сборка на Linux (Fedora)

### Требования
- GCC 10+ или Clang 11+
- C++17 поддержка

### Команды сборки
```bash
cd CalculatorFractions
g++ -std=c++17 -Wall -Wextra -I src src/UFrac.cpp src/UEditor.cpp src/UControl.cpp src/main.cpp -o CalculatorFractions
./CalculatorFractions
```

## Сборка на Windows (Visual Studio 2022 / C++ Builder)

### Для C++ Builder
1. Откройте C++ Builder
2. Создайте новый VCL Forms Application
3. Добавьте файлы из папки `src/` в проект
4. Создайте форму с компонентами согласно UClcPnl.h
5. Скомпилируйте проект

### Компоненты формы
- **TStaticText** (2 шт.): DisplayLabel, MemoryStatusLabel
- **TBitButton** (22 шт.): кнопки цифр, операций, функций, памяти
- **TMainMenu**: Правка (Копировать, Вставить), Вид (Дробь, Число), Справка (О программе)

### Расположение кнопок (стиль Windows Calculator)

## Поддерживаемые операции

### Арифметические операции
- `+` Сложение
- `-` Вычитание
- `*` Умножение
- `/` Деление

### Функции
- `Sqr` Возведение в квадрат
- `Rev` Обратное значение (1/x)

### Команды редактирования
- `C` Сброс калькулятора
- `CE` Очистка текущего ввода
- `←` (Backspace) Удаление последнего символа
- `+/-` Смена знака
- `/` или `|` Разделитель дроби

### Память
- `MC` Очистить память
- `MS` Сохранить в память
- `MR` Восстановить из памяти
- `M+` Добавить к памяти

### Буфер обмена
- Копировать (Ctrl+C)
- Вставить (Ctrl+V)

## Формат ввода дробей

Дроби вводятся в формате: `[-]<числитель>/<знаменатель>`

Примеры:
- `5/1` - целое число 5
- `3/4` - дробь три четверти
- `-2/3` - отрицательная дробь
- `0/1` - ноль

## Режимы отображения

1. **Дробь** - результат всегда отображается как дробь
2. **Число** - если знаменатель = 1, отображается как целое число

## Примеры вычислений

### Простая операция
```
5/1 + 2/1 = 7/1
```

### Операция с одним операндом
```
5/1 * = 25/1
```

### Повторное выполнение
```
5/1 + 4/1 = 9/1 = 13/1 = 17/1
```

### Вычисление функций
```
5/1 Sqr = 25/1
```

### Сложное выражение
```
6/1 Sqr + 2/1 Sqr / 10/1 + 6/1 = 10/1
```

## Тестирование

Консольная версия включает набор тестов для всех модулей:
- TFrac - арифметические операции с дробями
- TEditor - ввод и редактирование
- TMemory - работа с памятью
- TProc - выполнение операций
- TCtrl - управление калькулятором
