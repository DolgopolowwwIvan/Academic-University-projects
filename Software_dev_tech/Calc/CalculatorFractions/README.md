# Калькулятор простых дробей (C# Windows Forms)

## Описание
Калькулятор для выполнения вычислений над простыми дробями с графическим интерфейсом на C# и Windows Forms.
Реализован в соответствии с заданием на объектно-ориентированное программирование.

## Архитектура проекта

### Модули

| Модуль | Класс | Описание |
|--------|-------|----------|
| TFrac.cs | TFrac | Класс "Простая дробь" |
| TEditor.cs | TEditor | Ввод и редактирование дробей |
| TMemory.cs | TMemory<T> | Параметризованная память |
| TProc.cs | TProc<T> | Параметризованный процессор |
| TCtrl.cs | TCtrl | Управление калькулятором |
| MainForm.cs | MainForm | Интерфейс (Windows Forms) |

## Сборка на Windows (Visual Studio 2022)

### Требования
- Visual Studio 2022
- .NET 6.0 SDK или новее
- Windows 10/11

### Команды сборки
```bash
cd CalculatorFractions
dotnet build
dotnet run
```

### Через Visual Studio
1. Откройте файл `CalculatorFractions.csproj` в Visual Studio 2022
2. Build → Build Solution (Ctrl+Shift+B)
3. Debug → Start Debugging (F5)

Executable файл будет создан в папке `bin/Debug/net6.0-windows/` или `bin/Release/net6.0-windows/`

## Поддерживаемые операции

### Арифметические операции
- `+` Сложение
- `-` Вычитание
- `*` Умножение
- `/` Деление

### Функции
- `Sqr` Возведение в квадрат
- `Rev` Обратное значение (1/x)

### Команды редактирования
- `C` Сброс калькулятора
- `CE` Очистка текущего ввода
- `←` (Backspace) Удаление последнего символа
- `+/-` Смена знака
- `/` или `|` Разделитель дроби

### Память
- `MC` Очистить память
- `MS` Сохранить в память
- `MR` Восстановить из памяти
- `M+` Добавить к памяти

### Буфер обмена
- Копировать (Ctrl+C)
- Вставить (Ctrl+V)

## Формат ввода дробей

Дроби вводятся в формате: `[-]<числитель>/<знаменатель>`

Примеры:
- `5/1` - целое число 5
- `3/4` - дробь три четверти
- `-2/3` - отрицательная дробь
- `0/1` - ноль

## Режимы отображения

1. **Дробь** - результат всегда отображается как дробь
2. **Число** - если знаменатель = 1, отображается как целое число

## Примеры вычислений

### Простая операция
```
5/1 + 2/1 = 7/1
```

### Операция с одним операндом
```
5/1 * = 25/1
```

### Повторное выполнение
```
5/1 + 4/1 = 9/1 = 13/1 = 17/1
```

### Вычисление функций
```
5/1 Sqr = 25/1
```

### Сложное выражение
```
6/1 Sqr + 2/1 Sqr / 10/1 + 6/1 = 10/1
```
