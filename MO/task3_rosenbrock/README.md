# Задача 3: Метод Розенброка

## Описание

Метод Розенброка — это метод прямого поиска с адаптивным обновлением направлений поиска. На каждой итерации система ортогональных направлений пересчитывается с использованием процедуры ортогонализации Грама-Шмидта.

## Алгоритм

1. Задается начальная точка x⁰ и система ортогональных направлений S₁⁰, S₂⁰, ..., Sₙ⁰
2. Последовательная минимизация функции по каждому направлению:
   xᵢ⁰ = xᵢ₋₁⁰ + λᵢSᵢ⁰, где λᵢ = arg min f(xᵢ₋₁⁰ + λSᵢ⁰)
3. После завершения этапа вычисляются новые направления:
   - Находятся векторы Aᵢ = λᵢSᵢ + λᵢ₊₁Sᵢ₊₁ + ... + λₙSₙ
   - С помощью процедуры Грама-Шмидта строится новая система ортогональных направлений
4. Первый вектор направляется вдоль общего перемещения на предыдущем шаге
5. Процесс повторяется

## Обновление направлений (Грама-Шмидт)

```
S₁ᵏ⁺¹ = A₁ᵏ / ||A₁ᵏ||

B₂ᵏ = A₂ᵏ - (A₂ᵏ)ᵀS₁ᵏ⁺¹ · S₁ᵏ⁺¹
S₂ᵏ⁺¹ = B₂ᵏ / ||B₂ᵏ||

Bₗᵏ = Aₗᵏ - Σ(Aₗᵏ)ᵀSₘᵏ⁺¹ · Sₘᵏ⁺¹  (m = 1..l-1)
Sₗᵏ⁺¹ = Bₗᵏ / ||Bₗᵏ||  (l = 2..n)
```

## Достоинства и недостатки

### Достоинства:
- Самый эффективный из трех методов
- Адаптивное обновление направлений поиска
- Хорошо справляется с овражными функциями
- Быстрая сходимость вблизи минимума

### Недостатки:
- Более сложная реализация
- Требует обработки вырожденных случаев (нулевые λ)
- Большее вычислительных затрат на каждой итерации
