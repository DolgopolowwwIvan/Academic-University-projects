```
f₁(x) = 10(x₁ + x₂ - 10)² + (x₁ - x₂ + 4)²
Начальная точка: x⁰ = [0, 0]ᵀ

```

## Краткое описание методов

### Метод Гаусса (покоординатный спуск)

Последовательно минимизирует функцию по каждой координате при фиксированных остальных. После прохода по всем координатам процесс повторяется.

**Достоинства:** простота, отсутствие производных  
**Недостатки:** медленная сходимость для овражных функций

### Метод Хука-Дживса

Комбинирует исследующий поиск вокруг базисной точки с поиском по "образцу" в направлении убывания функции.

**Достоинства:** эффективнее Гаусса, адаптивный шаг  
**Недостатки:** требует настройки параметров

### Метод Розенброка

Использует адаптивную систему ортогональных направлений, которая обновляется на каждой итерации с помощью процедуры Грама-Шмидта.

**Достоинства:** самый эффективный, хорош для овражных функций  
**Недостатки:** сложная реализация
