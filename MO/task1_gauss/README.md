# Задача 1: Метод Гаусса (покоординатный спуск)

## Описание

Метод Гаусса — это метод прямого поиска (0-го порядка) для безусловной минимизации функции нескольких переменных. Основная идея заключается в последовательной минимизации функции по каждой координате при фиксированных значениях остальных.

## Алгоритм

1. Задается начальная точка x⁰ = (x₁⁰, x₂⁰, ..., xₙ⁰)ᵀ
2. На первой итерации ищется минимум функции при изменяющейся первой координате:
   x₁¹ = arg min f(x₁, x₂⁰, ..., xₙ⁰)
3. Получается новая точка x¹ = (x₁¹, x₂⁰, ..., xₙ⁰)
4. Из точки x¹ ищется минимум функции, изменяя вторую координату:
   x₂¹ = arg min f(x₁¹, x₂, x₃⁰, ..., xₙ⁰)
5. Процесс продолжается для всех координат
6. После n шагов процесс возобновляется с первой переменной

## Условия остановки

- f(xᵏ⁺¹) - f(xᵏ) ≤ ε₀
- |xᵢᵏ⁺¹ - xᵢᵏ| ≤ εᵢ, ∀i

## Тестовые функции

1. **Квадратичная функция:**
   - f₁(x) = 10(x₁ + x₂ - 10)² + (x₁ - x₂ + 4)²
   - Начальная точка: x⁰ = [0, 0]ᵀ

2. **Функция Розенброка:**
   - f₂(x) = 100(x₂ - x₁²)² + (1 - x₁)²
   - Начальная точка: x⁰ = [-1.2, 1]ᵀ

## Достоинства и недостатки

### Достоинства:
- Простота реализации
- Не требует вычисления производных
- Гарантирует сходимость для выпуклых функций

### Недостатки:
- Медленная сходимость для овражных функций
- Зависимость от выбора начальной точки
- Возможность "застревания" на плоских участках
