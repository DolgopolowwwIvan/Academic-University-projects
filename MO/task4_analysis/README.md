# Задача 4: Сравнительный анализ методов

## Описание

Данная задача включает в себя:
1. Сравнение всех трех методов (Гаусса, Хука-Дживса, Розенброка)
2. Исследование сходимости алгоритмов в зависимости от точности
3. Анализ траекторий спуска из различных начальных приближений
4. Формулирование выводов о достоинствах и недостатках методов

## Использование

```bash
cd task4_analysis
python comparison.py
```

## Что делает скрипт

### 1. Сравнительный анализ для квадратичной функции
- Запускает все три метода на квадратичной функции
- Строит сравнительные графики траекторий
- Строит графики сходимости

### 2. Сравнительный анализ для функции Розенброка
- Запускает все три метода на функции Розенброка
- Строит сравнительные графики траекторий
- Строит графики сходимости

### 3. Исследование сходимости при разных точностях
- Тестирует методы при ε = 10⁻², 10⁻⁴, 10⁻⁶, 10⁻⁸
- Строит график зависимости числа итераций от точности

### 4. Анализ из разных начальных точек
- Тестирует каждый метод из нескольких начальных точек
- Сравнивает результаты

## Основные выводы

### Метод Гаусса
**Достоинства:**
- Простота реализации
- Не требует вычисления производных
- Гарантирует сходимость для выпуклых функций

**Недостатки:**
- Медленная сходимость для овражных функций
- Зависимость от выбора начальной точки
- Возможность "застревания" на длинных плоских участках

### Метод Хука-Дживса
**Достоинства:**
- Эффективнее метода Гаусса за счет поиска по образцу
- Адаптивный выбор шага
- Хорошая сходимость на гладких функциях

**Недостатки:**
- Может требовать настройки параметров (delta, alpha, beta)
- На некоторых функциях может быть медленным

### Метод Розенброка
**Достоинства:**
- Самый эффективный из трех методов
- Адаптивное обновление направлений поиска
- Хорошо справляется с овражными функциями
- Быстрая сходимость вблизи минимума

**Недостатки:**
- Более сложная реализация
- Требует обработки вырожденных случаев

## Рекомендации

- Для простых квадратичных функций можно использовать любой метод
- Для сложных овражных функций (как функция Розенброка) предпочтительнее метод Розенброка или Хука-Дживса
- Метод Гаусса подходит для начального грубого поиска
- Выбор точности одномерного поиска существенно влияет на сходимость
